# Prompt Engineering — Коллекция промптов для AI агентов

Коллекция мета-промптов и шаблонов для оптимизации запросов к AI агентам по разработке ПО.

## 📚 Содержимое

| Файл | Описание |
|------|----------|
| [PromptGen.md](PromptGen.md) | Мета-промпт для преобразования сырых запросов в структурированные промпты production-качества |

## 🚀 Быстрый старт

### Использование PromptGen

1. Откройте [PromptGen.md](PromptGen.md)
2. Заполните секцию **[ВАШ ЗАПРОС]** своими требованиями
3. Отправьте AI агенту для генерации оптимизированного промпта
4. Используйте сгенерированный промпт для реализации задачи

```bash
# Скопируйте PromptGen.md в ваш проект
cp PromptGen.md /path/to/your/project/
```

## 🎯 Для чего это нужно

- ✅ Получение кода production-качества за один проход
- ✅ Снижение количества итераций и уточнений
- ✅ Стандартизация запросов в команде
- ✅ Версионирование промптов вместе с кодом
- ✅ Снижение фрустрации при работе с AI агентами

## 🏗️ Структура оптимизированного промпта

PromptGen генерирует промпты со следующей структурой:

- **Контекст и цель** — какую проблему решаем
- **Критерии приёмки** — измеримые требования (Definition of Done)
- **Архитектурные требования** — доменная модель, паттерны, стек
- **Требования к реализации** — функциональные и нефункциональные
- **Стратегия тестирования** — unit, integration, E2E тесты
- **Безопасность** — чек-лист security-проверок
- **Примеры использования** — specification by example
- **Self-verification** — AI проверяет себя перед выдачей

## 💡 Лучшие практики

| Практика | Описание |
|----------|----------|
| **Начинайте с нового чата** | Без prior context для чистоты выполнения |
| **Будьте конкретны** | Измеримые требования = лучший результат |
| **Добавляйте примеры** | Specification by example работает лучше описаний |
| **Используйте self-verification** | Заставьте AI проверить себя |
| **Итеративно улучшайте** | Сохраняйте успешные промпты как шаблоны |
